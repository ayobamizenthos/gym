# 6:45 Fitness - Gym Website

This is a React-based gym website built with Vite, TypeScript, and Tailwind CSS.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
